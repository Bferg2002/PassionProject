# PassionProject

## Elevator Pitch
ShellSchedule is a specialized full-stack pet care dashboard designed for chelonian (tortoise and turtle) owners to manage the health, habitat conditions, and ongoing costs of long-lived reptiles. The application features automated husbandry countdown alerts, an explicit category-based expense ledger, and an integrated herpetology AI assistant powered by Gemini to provide immediate, science-based species care data.

## Problem Statement
Reptiles have exceptionally long lifespans and strict, highly sensitive environmental requirements, yet mainstream pet applications focus entirely on dogs and cats. Tortoise keepers struggle with decentralized data when monitoring crucial health trends like linear shell growth, calculating invisible UVB bulb degradation timelines, and tracking the steep financial commitments of exotic vet care and habitat equipment. ShellSchedule solves this by consolidating growth analytics, specialized budgeting tools, and an immediate, domain-specific AI expert into a single, cohesive desktop workspace.

## Target User
Tortoise and Reptile owners in general would love to use this app as a way to keep track of everything related to their pet in one central location. This app would help with staying organized and keeping up to date with your reptiles need. 

## Why This Project
As a developer deeply interested in herpetology, I know firsthand that niche animal keeping requires highly specific tracking parameters that general-consumer technology completely overlooks. Building ShellSchedule allows me to merge my personal passion for reptile preservation with an enterprise-grade tech stack. This project serves as a meaningful utility that showcases clean component architecture, dynamic state layouts, external AI model integrations, and structured relational data management.
